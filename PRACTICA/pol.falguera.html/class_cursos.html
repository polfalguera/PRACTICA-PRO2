Toma nota del resultado de un envío hecho por un usuario a un problema, actualizando a la vez todo lo que conlleva el envío en si y que éste resulte exitoso o no.

Precondición
Cierto
Postcondición
Sea un envío exitoso o no, se actualizan las estadísticas del usuario con identificador u y del problema con identificador p. Además, se actualizan los problemas resueltos y enviables del usuario y se comprueba que con este envío el usuario haya completado o no el curso en el que estaba inscrito

void Cursos::listar_cursos ( ) const

Operación de escritura.

Precondición
Cierto
Postcondición
Se imprimen por pantalla los cursos del conjunto de forma creciente por su identificador. Para cada uno, se muestra el número de usuarios actuales o pasados que lo han completado, el número de usuarios inscritos actualmente y el número de sesiones que lo forman y los identificadores de dichas sesiones, en el mismo orden que se leyeron cuando se creó el curso
